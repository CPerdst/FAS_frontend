# FAS_frontend

环境变量：
- VITE_BACKEND_HOST - 后端服务器ip地址 - 仅测试时vite服务器使用
- VITE_BACKEND_PORT - 后端服务器端口 - 仅测试时vite服务器使用
- VITE_BACKEND_BASE_URL - 后端服务器的接口处理根路径 - 仅测试时vite服务器使用
- VITE_HOST - 开发服务器地址
- VITE_PORT - 开发服务器端口
- VITE_BASE_URL - 开发服务器根路径
